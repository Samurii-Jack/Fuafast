# FuaFast Website

A high-performance, mobile-first Single Page Application (SPA) for FuaFast Laundry Delivery Service in Ngong, Kenya.

## Features
- **Mascot-Led Design**: Character-led guiding experience.
- **Responsive**: Full Desktop, Tablet, and Mobile support.
- **Interactive**: Scroll-triggered animations, price counters, and bubble effects.
- **Conversion Focused**: One-click WhatsApp and Call CTAs.

## Tech Stack
- Vanilla HTML5
- Vanilla CSS3 (Custom Design System)
- Vanilla JavaScript

## How to Run
Simply open `index.html` in any modern web browser.
